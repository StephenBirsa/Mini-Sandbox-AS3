Controls For Mini Sandbox:
-Left Click = Destroy Block
-Right Click = Place Block
-WAD = Move Left, Right and Jump
-LEFT ARROW KEY + RIGHT ARROW KEY = change chunks quicker

//Experimental(Chance Of Being Buggy):
-K = save game to sharedobject (appdata)
-L = load game from sharedobject (appdata)

Manually Change chunks:
-walk to either left or right end of stage and you will be sent to the next chunk

Note:
1 chunk is 2.6 kb
200 chunks is 492 kb
1000 chunks is 2.4 mb
5000 chunks is about 12 mb
!Beware of SIZE INCREASE when saving more chunks!
